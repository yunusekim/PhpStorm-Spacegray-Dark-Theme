# PhpStorm Spacegray-Dark Theme
Spacegray-Dark Colorscheme for PhpStorm.

## Installation
* Copy `Spacegray-Dark.icls` to your PhpStorm Preferences:
	- On Mac it is in `~/Library/Preferences/WebIde<version>0/colors`.
	- On Linux it is in `/home/<username>/.WebIde<version>0/config/colors`.
	- On Windows it is in `C:\Users\<username>\.WebIde<version>0\config\colors`.

	Replace `WebIde<version>0` with the Version of PhpStorm you use, for example: WebIde80 for PhpStorm 8.

* Restart `PhpStorm`.

* Go to `Preferences > Editor > Colors & Fonts` and select one of the new color themes.
